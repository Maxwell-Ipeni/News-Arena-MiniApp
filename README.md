# News-Arena-MiniApp by MG

A responsive news web application that delivers personalized and reliable news content. Built with Bootstrap 5 and Vanilla JavaScript.

## Features

- **Category Filters:** Browse news by categories (General, Business, Entertainment, Health, Science, Sports, Technology)
- **Search Functionality:** Search for articles by keywords
- **Like & Bookmark:** Interact with news articles
- **Responsive Design:** Works on desktop, tablet, and mobile devices
- **Featured Carousel:** Top news stories displayed in a carousel

## Getting Started

### Prerequisites

- Modern web browser
- Internet connection (required for Bootstrap, Font Awesome, and MediaStack API)

### Clone the Repository

```bash
git clone https://github.com/Maxwell-Ipeni/News-Arena-MiniApp.git
```

### Run the Application

1. Navigate to the project directory:

```bash
cd News-Arena-MiniApp
```

2. Open `index.html` directly in your browser, OR serve locally:

```bash
# Using Python
python3 -m http.server 8000
```

3. Open your browser and visit:

```
http://localhost:8000
```

## Usage

- **Browse Categories:** Click on category links in the navigation bar
- **Search:** Click the search icon to open the search modal
- **Like Articles:** Click the thumbs-up icon on any article
- **Bookmark:** Click the bookmark icon to save articles
